# Seja bem vindo ao linux-troubleshooting-lab

---

## O que é isso?

Este projeto será voltado para aprofundar os conhecimentos em linux, tratando fundamentos mais profundos do S.O

---

## O que vou aprender?

Vamos aprender na prática, como realizar o Troubleshooting de um sistema operacional linux, aprofundar nos comandos, gerenciamento
de processos, hierarquias e permissões, redes (ip, ss, netstat), gerenciamento de serviços.
